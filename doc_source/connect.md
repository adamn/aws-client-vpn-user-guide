# Connect using an OpenVPN client<a name="connect"></a>

You can connect to a Client VPN endpoint using common Open VPN client applications\.

**Note**  
For SAML\-based federated authentication, you must use the AWS provided client to connect to a Client VPN endpoint\. For more information, see [Connect using an AWS provided client](connect-aws-client-vpn-connect.md) or contact your VPN administrator\.
